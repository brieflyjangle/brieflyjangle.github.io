An **interval** is the ==distance between two [[Music-Notes|notes]]==. Intervals are the building blocks for understanding [[Music-Chords|chords]] and scales.

<!-- more -->

## Measuring Intervals

Intervals are measured in **half steps** (semitones) and **whole steps** (tones):

- **Half step** = the smallest distance between two [[Music-Notes|notes]] (e.g. `C` to `C#`)
- **Whole step** = two half steps (e.g. `C` to `D`)

## Common Intervals

| Interval Name     | Half Steps | Example (from C) |
|:------------------|:----------:|:------------------|
| Unison            | 0          | C → C             |
| Minor 2nd         | 1          | C → C#            |
| Major 2nd         | 2          | C → D             |
| Minor 3rd         | 3          | C → Eb            |
| Major 3rd         | 4          | C → E             |
| Perfect 4th       | 5          | C → F             |
| Tritone           | 6          | C → F#            |
| Perfect 5th       | 7          | C → G             |
| Minor 6th         | 8          | C → Ab            |
| Major 6th         | 9          | C → A             |
| Minor 7th         | 10         | C → Bb            |
| Major 7th         | 11         | C → B             |
| Octave            | 12         | C → C (next)      |

## Consonance vs Dissonance

> [!tip] Quick Rule
> - **Consonant** intervals sound stable and pleasant (unison, 3rds, 5ths, octave)
> - **Dissonant** intervals sound tense and want to resolve (2nds, 7ths, tritone)

## Related Concepts

- Intervals measure the distance between [[Music-Notes|Notes]]
- Stacking intervals (specifically 3rds) is how [[Music-Chords|Chords]] are built
